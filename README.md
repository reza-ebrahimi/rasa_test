### Setup
```console
$ git clone https://github.com/reza-ebrahimi/rasa_venv.git
$ cd rasa_venv
$ tar -xvzf archive/venv-tensorflow.tar.gz venv
```

### activate environment
```console
$ source venv/bin/activate
(venv) $
```

### deactivate environment
```console
(venv) $ deactivate
```
